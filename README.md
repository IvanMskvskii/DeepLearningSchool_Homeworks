# DeepLearningSchool_Homeworks
Школа глубокого обучения (Deep Learning School) – учебная организация на базе Физтех-школы прикладной математики и информатики Московского физико-технического института.

Первый семестр посвящен общей теории нейронных сетей, а также нейронным сетям в компьютерном зрении.
Второй семестр посвящен использованию нейросетевых моделей для обработки естественного языка (NLP).
